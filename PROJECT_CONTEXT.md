# Проект: GemCraft — Казуальная игра для VK Games

## Дата начала обсуждения: 2026-03-08
## Последнее обновление: 2026-03-10

---

## 1. Общее описание проекта

**Рабочее название:** GemCraft (финальное — определить позже)

**Цель:** Создать казуальную 2D match-3 игру для платформы VK Games с доходом от 100 000 руб/мес.

**Связь с бизнесом:** Группа **Sunroom** (ВКонтакте, продажа браслетов) — стартовая база для привлечения игроков. Активная аудитория для рассылки: **100 000+ человек**.

**Ключевое требование:** Игра должна органично ложиться на аудиторию Sunroom (тематика камней, минералов, самоцветов), но при этом **не иметь жёсткой привязки** к Sunroom и работать как самостоятельный продукт для широкой аудитории.

**Сеттинг:** Игрок — мастер-ювелир, нашедший заброшенную мастерскую в горах. Восстанавливает её, собирает коллекцию минералов, становится легендарным мастером. Фоновый нарратив — **Всемирная Гильдия Ювелиров** присылает письма с заданиями, подарками и поддержкой.

## 2. Архитектура

### Модульный принцип: мета отделена от core loop

```
[МЕТА-СЛОЙ]              ←→  [CORE LOOP]
Мастерская (строительство)     Match-3 (текущий)
Коллекция камней               Bubble Shooter (запасной)
Battle Pass                     Любой другой (подменяемый)
Магазин
Лидерборд
```

**Контракт между модулями:**
- Core loop получает: конфигурацию уровня (цели, ограничения, типы камней, блокеры)
- Core loop возвращает: результат (пройден/нет, звёзды, собранные камни по типам, очки)
- Мета не знает, КАК игрок собирал камни — только ЧТО он собрал

При такой архитектуре можно в любой момент подменить core loop механику без переделки меты.

## 4. Выбор core loop и отклонённые идеи

### Выбрано: Match-3

Причины:
- Максимальный потенциал левел-дизайна (формы поля, блокеры, спецкамни, цели)
- Проверенная монетизация через ходы
- Огромная аудитория жанра
- Отличная совместимость с метой (задания «собери N камней» = естественные цели)

Запасной вариант: Bubble Shooter (можно подключить как отдельный модуль благодаря модульной архитектуре).

### Отклонённые идеи:

| Идея | Причина отказа |
|---|---|
| Merge | Сложная визуализация эволюции камней |
| GemPop (кликер-пазл) | Отклонён |
| Crystal Lines | Не понравилась |
| Stone Tower (Idle) | Склоняемся к match-3, а не к idle/кликерам |
| Rune Match (Match-3 + RPG) | Не подходит тематика боёв, монстров, героев |
| Gem Blast (Zuma) | Слабая мета-совместимость, тяжело разнообразить после 50+ уровней |
| Crystal Solitaire (Маджонг) | Тяжело масштабируется, мелкие элементы неудобны на телефоне |
| Gem Catcher (аркада) | Нет левел-дизайна |

## 5. Core Loop: Match-3 — детальное описание

### Базовая механика:
- Поле 7x7 или 8x8, заполненное камнями 5-6 типов (цветов)
- Свайп камня — меняется местами с соседним
- **Если комбинация не собралась — камни возвращаются на место**
- 3+ одинаковых в ряд → уничтожаются, сверху падают новые
- **Ограничение: количество ходов** (щедрое, 30-40+ ходов, большинство уровней проходимы с первого раза)

### Дизайн уровней:
- Основная масса уровней — **проходимые** с первого раза
- Есть специальные **фейловые уровни**, где игрок по плану должен проиграть (точки монетизации: продолжение за рекламу/монеты)
- Соотношение примерно: каждый 5-7 уровень = фейловый

### Типы целей уровней:

**Правило: максимум 2 цели на уровень.** 4 типа целей во всей игре.

| Тип цели | Иконка на плашке | Что делать |
|---|---|---|
| Собери N камней цвета | Маленький камень нужного цвета + число | Уничтожь N камней этого цвета |
| Разбей лёд | Иконка льда + число | Освободи N камней из льда |
| Опусти предмет | Иконка предмета + стрелка вниз | Проведи предмет до нижнего ряда |
| Набери очки | Иконка звезды + число | Набери N очков |

**Визуализация целей:** Компактная плашка вверху экрана: иконка + числовой прогресс (например, `[рубин] 12/20`). Максимум 2 цели + счётчик ходов. Должно считываться за 1 секунду.

### Спецкамни:

**Создание:** При комбинации 4+ камней → обычные камни исчезают → на месте камня, которым игрок свайпнул, появляется спецкамень. Спецкамень **НЕ активируется сразу**, а остаётся на поле.

**Активация:** Спецкамень активируется, когда игрок включает его в следующую комбинацию (свайпает его в группу 3+). Исключение: радужный камень активируется свайпом на любого соседа.

| Комбинация | Спецкамень | Визуал | Эффект при активации |
|---|---|---|---|
| 4 в ряд (горизонталь) | Горизонтальный полосатый | Камень того же цвета + горизонтальная полоса | Уничтожает весь ряд |
| 4 в ряд (вертикаль) | Вертикальный полосатый | Камень того же цвета + вертикальная полоса | Уничтожает весь столбец |
| L или T-образная (5 камней) | Камень-бомба | Камень того же цвета + пульсирующее свечение | Взрывает область 3x3 |
| 5 в ряд | Радужный камень | Переливающийся, без цвета | Уничтожает ВСЕ камни одного цвета (того, на который свайпнули) |

**Комбо двух спецкамней** (свайп двух спецкамней друг на друга):

| Комбинация | Эффект |
|---|---|
| Полосатый + Полосатый | Крест (ряд + столбец) |
| Полосатый + Бомба | 3 ряда + 3 столбца |
| Бомба + Бомба | Взрыв 5x5 |
| Радужный + Полосатый | Все камни цвета полосатого → превращаются в полосатые и срабатывают |
| Радужный + Бомба | Все камни одного случайного цвета → превращаются в бомбы и срабатывают |
| Радужный + Радужный | Очищает всё поле |

**Предустановленные спецкамни:** Левел-дизайнер может разместить спецкамень на поле до начала уровня (инструмент дизайна).

### Блокеры:

| Блокер | Механика | Визуализация |
|---|---|---|
| **Лёд (1-3 слоя)** | Камень заморожен, нельзя двигать. Каждая комбинация рядом снимает 1 слой | 1 слой = полупрозрачный, 2 = плотный, 3 = непрозрачный. Трещины при каждом ударе |
| **Камень / Горная порода** | Неразрушимый блок, занимает клетку. Формирует форму поля | Тёмный грубый булыжник, очевидно «не игровой» |
| **Опускание предмета** | Целевой предмет (ключ, осколок), не участвует в комбинациях. Падает вниз при расчистке под ним. Цель = довести до нижнего ряда | Визуально отличается от камней (золотой ключ / светящийся осколок). Стрелка вниз при первом появлении |

**Сейф** — отложен на следующие итерации.

Каждая новая локация на карте вводит 1-2 новых блокера/механику.

### Бустеры:

**Внутриуровневые** (активируются во время игры, покупаются за монеты):
1. **Молот** — уничтожает 1 камень по выбору
2. **Перемешивание** — случайно перемешивает все камни на поле
3. **+5 ходов** — добавляет ходы на текущем уровне

**Предстартовые** (активируются перед началом уровня, покупаются за монеты):
1. **Радужный камень** — на поле появляется 1 радужный камень в случайной позиции
2. **Бомба на старте** — на поле появляется 1 камень-бомба в случайной позиции

### Система оценки:
- 1 звезда = уровень пройден (минимальная цель выполнена)
- 2 звезды = выполнена доп. цель
- 3 звезды = идеальное прохождение
- Количество звёзд определяет **количество и качество ресурсов**:
  - 1 звезда → 1 случайный ресурс
  - 2 звезды → 2 ресурса
  - 3 звезды → 3 ресурса + шанс на редкий

## 6. Технологический стек

**Решение: Phaser 3 + TypeScript**

Причины:
- VK Games = VK Mini Apps (веб-приложение в iframe)
- Unity WebGL-билды слишком тяжёлые (10-50 МБ) — убивают конверсию на мобильных
- Phaser 3: лёгкий билд (500 КБ - 1 МБ), встроенные системы анимаций и физики
- Хорошо генерируется через ИИ (Claude Opus 4.6)
- Графика: генерация через Nano-Banano + удаление фона

**Опыт команды:** Unity-разработчик (полезно для понимания игровой логики).

## 7. Монетизация

### Реклама:
- **Rewarded Video** — основной рекламный формат:
  - Продолжение уровня после проигрыша (+5 ходов). Техническое требование: должна быть возможность «продолжить» уровень, а не перезапустить
  - Удвоение наград за уровень
  - Ежедневный сундук
  - Бесплатные монеты (кнопка «Получить 100 монет» раз в 4 часа)
- **Interstitial — ОТКЛЮЧЁН на первой итерации** для улучшения Retention и LTV
- VIP-подписка — **НЕ будет**

### Внутриигровой магазин (VK Pay):
- Монеты (паки: 500 / 2000 / 5000)
- Бустеры
- Пак ресурсов (дерево + камень + стекло)
- Battle Pass (премиальная дорожка)

### Battle Pass:
- Бесплатная дорожка + премиальная (за покупку)
- Прогресс: прохождение уровней + выполнение миссий из Battle Pass
- Визуализация в мастерской: **доска заданий на стене**
- Сезонный (30 дней), с уникальными наградами

### Стартовый бонус от Sunroom:
- При переходе из группы Sunroom (отслеживание через `vk_ref` параметр VK Mini Apps) игрок получает:
  - 500 монет
  - 3 бустера (молот, перемешивание, +5 ходов)
  - **Камень Sunroom** в коллекцию — эпический, +10% к монетам (лучше обычных камней на монеты, которые дают +3-5%)

## 9. Метапрогрессия

### 9.1. Мастерская — ОСНОВА МЕТЫ

**Главный экран = Мастерская.** Фронтальный вид (side view). Одно визуальное пространство с интерактивными объектами.

#### Ключевые принципы:
- **Никаких чисто декоративных объектов** — каждый объект либо функциональный (открывает механику), либо даёт улучшение при прокачке
- **Прокачал → получил улучшение** того, что прокачал (основное правило, редкие исключения допустимы)
- **5 интерактивных предметов на комнату** (+ возможны декоративные исключения без прокачки)
- **Магазин** — отдельная кнопка в UI, не привязан к объекту в комнате

#### Система «Комнат» (длинная мета-прогрессия):

Мастерская — это **растущее пространство**. Навигация между комнатами — определить после понимания итогового количества комнат.

На данный момент детально проработана только **Комната 1**. Остальные комнаты будут проектироваться после запуска и тестирования первой.

| Комната | Уровни | Статус |
|---|---|---|
| **1. Мастерская** | 1-50 | Детально проработана (см. ниже) |
| **2-5+** | 50+ | Будут проектироваться после тестирования Комнаты 1 |

#### Комната 1: Мастерская (уровни 1–50) — ФИНАЛЬНЫЙ СОСТАВ

| # | Объект | Тип | Функция по тапу | Прокачка |
|---|---|---|---|---|
| 1 | **Дверь** | Функциональный (без прокачки) | Открывает карту путешествий (выбор уровней) | Нет (исключение) |
| 2 | **Полка с минералами** | Функциональный + прокачка | Открывает коллекцию камней / управление слотами | Улучшение → дополнительные слоты для камней (старт: 1 слот → макс. ~5) |
| 3 | **Почтовый ящик** | Функциональный + прокачка | Письма Гильдии Ювелиров + ежедневный подарок (daily bonus) | Улучшение → увеличивается ценность ежедневных подарков |
| 4 | **Доска заданий** | Функциональный + прокачка | Открывает Battle Pass | Улучшение → больше очков Battle Pass за выполнение миссий |
| 5 | **Верстак** | Декоративный (исключение) | Нет функции (пока) | Нет (кандидат на будущую механику: огранка камней коллекции для усиления бонусов) |

**Дополнительно:**
- **Камин** — декоративный элемент (горит), функционал определится позже
- **Магазин** — кнопка в UI (не объект в комнате)

#### Прогрессия внутри комнаты:
- Предмет сначала **сломан** → чинишь за ресурсы → получаешь доступ к функции
- Функциональные предметы **прокачиваются** за ресурсы → каждый уровень даёт конкретное улучшение
- **Визуальных апгрейдов (деревянный → мраморный → золотой) НЕТ** — они конфликтуют со скинами
- Рейтинг мастерской = количество починенных/прокачанных предметов по всем комнатам

### 9.2. Коллекция минералов

**Расположение:** Полка/витрина внутри мастерской (не отдельный раздел меню).

**Механика слотов:**
- Игрок собирает камни (50+ со временем), каждый с уникальным бонусом
- **Активировать** можно только ограниченное количество одновременно:
  - Старт: **1 слот**
  - Дополнительные слоты открываются через **прокачку Полки с минералами** в мастерской
  - Максимум: **~5 слотов**
  - Возможно, 6-й слот = из Battle Pass (премиум)
- Игрок **выбирает**, какие камни активировать → решение = вовлечённость

**Бонусы от камней — только экономические (НЕ геймплейные, НЕ косметические):**
- +% к монетам за уровень
- +% к определённому ресурсу (дерево / камень / стекло)
- Сеты камней (3 из 5 одной группы) = одноразовый ресурсный пак

**Камни НЕ дают косметику** — это конфликтует с системой слотов (камень в слоте = бонус, а не скин). Косметика добывается из других источников: прогрессия, магазин, Battle Pass.

Это **не влияет на левел-дизайн** — можно добавлять камни бесконечно без проблем с балансом.

**Примеры камней:**

| Камень | Редкость | Бонус | Источник |
|---|---|---|---|
| Кварц | Обычный | +3% монет | Прохождение уровней |
| Аметист | Обычный | +5% к ресурсу «дерево» | Прохождение уровней |
| Рубин | Редкий | +8% монет | Сундук дня / Battle Pass |
| Камень Sunroom | Эпический | +10% монет | Только при переходе из Sunroom |
| Алмаз | Легендарный | +15% монет | Ивент / поздняя прогрессия |

### 9.3. Карта путешествий

- Открывается через **Дверь** в мастерской
- Разбита на локации: Горный рудник → Морская пещера → Вулкан → Подводные гроты → ...
- Каждая локация = 20-30 уровней + уникальный визуальный сет камней + новые блокеры/механики
- Линейная прогрессия (уровни открываются последовательно)

### 9.4. Battle Pass

- Визуализация: **доска заданий** на стене мастерской
- Бесплатная дорожка + премиальная
- Прогресс: прохождение уровней + миссии (собери 50 рубинов, пройди 3 уровня без бустеров и т.д.)
- Сезонный (30 дней)
- Награды: монеты, ресурсы, бустеры, камни в коллекцию, сезонные скины декора, возможно слот камня

### 9.5. Гильдия Ювелиров (нарратив)

- Фоновый нарратив через **письма в почтовом ящике**
- Обоснование: подарков, ивентов, Battle Pass («Сезонный заказ от Гильдии»), обучения
- Первые письма = мягкий туториал: «Добро пожаловать! Вот ваш первый заказ...»
- Не кат-сцены и не диалоги — просто короткие письма

### Принцип: «Матрёшечный» интерфейс
- Главный экран = одно пространство (мастерская)
- Все мета-элементы вложены как интерактивные объекты
- Никаких 5 кнопок в главном меню
- Между комнатами — свайп

---

## 10. Социальные функции

### Лидерборд мастерских:
- Рейтинг по числовому показателю **«Мастерство»** — одно число, видимое игроку. Формула скрыта, складывается из:
  - Количество собранных звёзд (суммарно по всем уровням)
  - Количество коллекционных камней
  - Количество починенных объектов в мастерской
- **Скины НЕ учитываются** в рейтинге
- Лидерборд друзей ВК (VK API) + глобальный

### Интерактивный просмотр чужих мастерских:
- Тап на игрока в лидерборде → **заходишь в его мастерскую**
- Можно свайпать между комнатами, тапнуть на полку → посмотреть коллекцию камней
- Read-only режим (нельзя ничего менять)
- Технически: загрузка конфига чужого игрока (набор ID предметов/скинов) → рендер на клиенте

## 11. Требования к аудитории и платформе

- **Основная платформа:** Мобильные устройства (телефоны) через ВКонтакте
- **Целевой DAU для 100К руб/мес:** 5 000 — 10 000
- **Целевой D1 Retention:** > 30%
- **Аудитория:** Преимущественно женская (аудитория Sunroom — покупательницы браслетов), 18-45 лет
- **Требование по UX:** Все интерактивные элементы должны быть удобны для тапа пальцем на телефоне

## 12. Ограничения и принципы

- **Все тексты в игре — исключительно на русском языке.** UI, кнопки, подсказки, письма Гильдии, Battle Pass, магазин — всё на русском
- **Только 2D.** Никакого 3D
- **Без боёв, монстров, героев.** Абстрактная, «мирная» тематика
- **Тематика:** Камни, минералы, самоцветы — универсальная, не привязанная жёстко к Sunroom
- **Графика:** Генерация через ИИ (Nano-Banano + удаление фона)
- **Код:** Генерация через Claude Opus 4.6
- **Мета не должна перегружать интерфейс** — матрёшечный подход
- **Модульность:** Core loop отделён от меты, можно подменять

## 13. Прогрессия игрока (таймлайн)

**Неделя 1 (уровни 1-30):**
- Простые match-3 уровни, обучение механикам
- Открывается первая локация на карте
- Начало восстановления мастерской (починка первых объектов)
- 3-5 камней в коллекции, 1 слот (прокачка полки → 2-й слот)

**Неделя 2-4 (уровни 30-100):**
- Новые блокеры (лёд, опускание предмета)
- 2-3 локации открыты
- Мастерская частично восстановлена, начинается декор
- 10-15 камней в коллекции
- Battle Pass активен

**Месяц 2+ (уровни 100+):**
- Сложные уровни с комбинацией блокеров
- Открывается 2-я комната (Витрина)
- 4-й слот камней
- Ивенты, сезонные обновления

**Месяц 3+ (уровни 200+):**
- 3-я комната (Сад)
- 5-й слот камней
- 30+ камней в коллекции
- Улучшение предметов (апгрейды)

## 14. Визуальный стиль — утверждённые решения

### Утверждённая концепция: «Горная мастерская» (Warm Craft)

| Параметр | Значение |
|---|---|
| **Стиль** | Мультяшный / Cartoon — яркий, контрастный, насыщенный, детально прорисованный |
| **Палитра** | Тёплая и яркая. Золото, янтарь, рубин, тёплый коричневый |
| **Камни на поле** | Круглые шарики, матовые (как каменные бусины), НЕ глянцевые |
| **Мастерская** | Уютная деревянная (бревенчатый домик в горах), фронтальный вид (side view) |
| **Фон поля** | Тематический (пещера с факелами, рудник — меняется по локациям) |
| **UI** | Скевоморфизм — деревянные таблички, каменные плашки, верёвочные элементы |
| **Связь с Sunroom** | Никакой визуальной связи |
| **Язык** | Все тексты — исключительно на русском |

Референсные изображения: `design/` (workshop/workshop.png, gameplay.png, map.png)
UI-спрайты: `design/workshop/ui_assets/`

### Отклонённые концепции:

| Концепция | Причина отказа |
|---|---|
| **B: Подземная кузница (Deep Mine)** | Слишком тёмная и «железная» атмосфера. Не соответствует тёплому уютному настроению |
| **C: Солнечная лавка (Sunny Workshop)** | Не понравился общий стиль. Слишком «лёгкий», не хватает контраста и насыщенности |

Подробный гайд по визуалу: `design/DESIGN_GUIDE.md`

### Структура кода:

```
src/

│   ├── GameScene.ts        # Match-3 геймплей + UI + модальные окна

│   └── SpecialGem.ts       # Логика эффектов спецкамней
├── systems/
│   ├── MatchSystem.ts      # Поиск матчей: 3+, L/T, 5-в-ряд

├── data/
│   └── levels.ts           # 15 уровней первой локации
└── utils/
    └── helpers.ts           # Утилиты
```
